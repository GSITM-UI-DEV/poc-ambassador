### 2. 로그인 `/login`
- 중앙 카드: 이메일/비밀번호, 자동 로그인 체크, [로그인] 골드 버튼
- 소셜 로그인 3종(카카오·네이버·Google) 버튼 — 목업 (클릭 시 동일 처리)
- 무엇을 입력해도 "홍길동 GOLD" 세션 생성 후 이전 페이지 복귀
- 우측(또는 하단)에 회원 혜택 안내 패널: "회원 전용가 최대 15%" 등 가치제안 노출

### 3. 검색 결과 `/search`
- 상단: 검색 조건 요약 바(지역·날짜·인원) + 변경 가능
- 좌측 필터: 목적 칩, 브랜드, 가격대, 편의시설(키즈풀·수영장·조식)
- 우측 리스트: 호텔 카드(이미지·태그·지역·**정가 취소선 + 회원가 + -15% GOLD 배지 + 포인트 적립**) — 제안서 KEY SCREEN ② 반영
- 정렬: 추천순/가격순/평점순
- 비로그인 시 회원가 blur + "로그인하고 회원가 보기" 유도 (가치제안 시연 포인트)

### 4. 호텔 상세 `/hotels/:hotelId`
- 이미지 갤러리 + 호텔 정보(등급·위치·편의시설)
- 탭: 객실 / 패키지 / 다이닝
- 객실·패키지 카드: 포함 내역 체크리스트, 정가/회원가/배지, [예약하기] → `/booking`
- Sticky 우측 예약 요약 패널(날짜·인원·선택 상품·합계)

### 5. 예약하기 `/booking` — 3-step 단일 페이지 (스텝 인디케이터)
- **STEP 1 옵션**: 패키지 포함 항목(잠김 표시) + 추가 옵션 토글(케이크·플라워·고층 요청·다이닝 시간)
- **STEP 2 혜택**: 회원 등급 카드(GOLD·보유 포인트), 회원 할인 자동 적용, 쿠폰 토글, 포인트 사용 입력 — 우측에 실시간 가격 계산 패널(정가→최종가 흐름)
- **STEP 3 결제**: 예약자 정보(자동 채움), 결제 수단(신용카드/카카오페이/네이버페이/해외카드), 취소 정책 안내, [○○○원 결제하기]
- 결제 클릭 → 1.5초 로딩 목업 → `/booking/complete`

### 6. 예약 완료 `/booking/complete`
- 체크 아이콘 + "예약이 완료되었습니다!" + 예약 번호(AMB-YYYYMMDD-XXXX)
- 예약 카드 요약, 적립 예정 포인트, Next Tier 게이지("PLATINUM까지 ○○P")
- 다음 액션: [예약 상세] [캘린더 추가] [다이닝 예약] — 관계 축 시연

### 7. 마이페이지 `/mypage` — 제안서 KEY SCREEN ④
- 상단: 회원명·등급 + **다음 등급 게이지 바**
- 위젯 4종: 보유 포인트 / 소멸 예정(⚠️ 60일 내 2,000P) / 쿠폰·바우처(사용하기) / 생애주기 추천(기념일 패키지)
- **예약 내역: TanStack Table** — 컬럼(예약번호·호텔·패키지·체크인·금액·상태·적립), 상태 탭 필터(전체/예약확정/이용완료/취소), 정렬
- 최근 본 숙소 카드 행

### 8. 스페셜 오퍼 `/offers`
- 카테고리 필터 칩(전체/기념일/가족/미식/비즈니스) + 호텔 필터
- 프로모션 카드 그리드(기간·혜택·가격·[예약하기])
- 상단에 "공식 채널 전용" 강조 배너

### 9. 페이지 URL 가이드 `/guide` — 내부용 인덱스
- 시연·검수용 단순 페이지. 디자인 공수 들이지 않음 (중앙 정렬 타이틀 + 기본 테이블)
- "AMBATEL 프로토타입 페이지 목록" 타이틀 아래 표 2개: PC / 모바일
- 컬럼: 페이지명 | URL (클릭 시 해당 페이지로 이동하는 링크)
- 레이아웃(헤더/푸터) 없이 독립 렌더

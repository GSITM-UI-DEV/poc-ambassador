# Google Flow (Veo) — 프롬프트 & 참조 파일 가이드

**THE AMBASSADOR SEOUL × GSITM — 40초 제안 필름**

Flow 화면 우측 프롬프트 박스("무엇을 만들고 싶으신가요?")에 아래 프롬프트를 **하나씩** 붙여넣어
클립을 생성하고, 내가 만든 UI 소재는 **업로드해서 그대로** 얹는 방식.

---

## 대원칙 (반드시 지킬 것)
- **UI 화면(clips/·stills/)은 업로드해서 그대로 배치** — 절대 Veo 생성 재료로 넣지 말 것 (한글·숫자 뭉개짐)
- **Veo 프롬프트로는 배경·트랜지션·무드만 생성** — 프롬프트마다 "글자·로고 없음" 명시
- **로고는 실제 파일을 오버레이** — Veo가 로고를 그리게 하지 말 것

## STEP 2 — 생성 프롬프트 (박스에 하나씩 붙여넣기)

각 프롬프트 = 약 8초 클립 1개. 화면비 16:9. 필요 컷만 생성하면 됨.

### ① 인트로 배경 플레이트 (S1 · 0–3초) — 이 위에 앰배서더 로고 올림
```
딥 차콜(#1c1b19) 배경 위로 미세한 골드빛(#b7935b) 입자들이 아주 느리게 떠다닌다.
화면 중앙을 향해 얇은 골드 빛줄기 하나가 우아하게 그려지듯 나타났다가 은은히 번진다.
소프트 포커스, 잔잔한 보케, 미세한 필름 그레인, 고요하고 격조 높은 럭셔리 호텔 무드.
카메라는 극도로 느린 줌인. 글자·문자·숫자·로고·워터마크 전혀 없음. 16:9, 시네마틱, 고해상도.
```

### ② 트랜지션 스팅 (씬 사이 전환용, 재사용)
```
어두운 화면을 가로질러 골드빛 입자와 부드러운 빛줄기가 왼쪽에서 오른쪽으로 빠르게 스윕한다.
짧고 우아한 빛의 전환, 모션블러, 럭셔리하고 미니멀. 글자·로고 없음.
16:9, 2초 내외의 임팩트 있는 트랜지션.
```

### ③ 무드 b-roll (선택 — 씬 사이 호흡용)
```
따뜻한 골드 조명이 감도는 추상적 럭셔리 텍스처. 실크 커튼의 결, 대리석 표면에 반사되는 빛,
샹들리에 보케가 아주 얕은 심도로 흐른다. 느린 카메라 드리프트, 시네마틱, 고요함.
사람·글자·로고 없음. 16:9.
```

### ④ 아웃트로 배경 플레이트 (S7 · 35–40초) — 이 위에 로고 락업 올림
```
딥 차콜 배경 중앙으로 골드빛 입자들이 부드럽게 모여들었다가 천천히 산개한다.
차분하게 리졸브되는 엔딩 무드, 은은한 골드 글로우, 미세한 그레인.
글자·로고 없음. 카메라 고정, 아주 느린 페이드아웃. 16:9, 시네마틱.
```

> 팁: 영어로 넣으면 디테일 반영이 더 정확할 때가 많음. 위 프롬프트를 영역별로 영어 번역해 써도 됨.
> 팁: 인트로 배경 → UI 클립으로 자연스럽게 넘길 땐 Flow의 first/last-frame 트랜지션에
> UI 스틸(`stills/pc-01-main.png`)을 '마지막 프레임'으로 지정하면 이음새가 부드러움.

## 출력 파일 명명 규칙 (`docs/video/result/`)
- **원본** (Flow에서 받은 그대로, 워터마크 있음): `flow_video1.mp4`, `flow_video2.mp4`, …
- **수정본** (워터마크 제거·1920×1080 완성, 실제 사용본): `flow_final_video1.mp4`, `flow_final_video2.mp4`, … (같은 번호끼리 매칭)
- 매핑(누적):
  | 번호 | 씬 | 원본 | 수정본 |
  |---|---|---|---|
  | 1 | ① 인트로 배경 (S1) | `flow_video1.mp4` | `flow_final_video1.mp4` ✅ |
  | 2 | ② 트랜지션 스팅 | `flow_video2.mp4` | `flow_final_video2.mp4` ✅ |

> Flow 무료 출력은 우하단에 ✦ 워터마크가 붙음 → 수정본은 크롭+1080p 업스케일로 제거.
> UI 녹화 클립(`clips/`)은 워터마크 없음(실제 화면 녹화). Flow 생성분만 이 처리 필요.

## STEP 3 — 조립 (타임라인)

`storyboard.md` 순서 그대로:
1. **① 인트로 배경** 위에 `amber-logo-light.png` 페이드인 + 자막 "디지털 경험을 다시 짓다"
2. `pc-01-main-hover.mp4` + 자막 "목적에서 시작하는 탐색"
3. `pc-03-detail-scroll.mp4` + 자막 "객실과 다이닝을, 한 번에"
4. **`pc-04-value-count.mp4`** (9초 배분·최장) + 자막 "GOLD 회원 15% · 공식 채널 전용가" → "338,000원"
5. `pc-05-mypage-gauge.mp4` + 자막 "관계를 설계하는 멤버십"
6. `mo-flow.mp4` (몽타주로 빠르게) + 자막 "손안에서 완성되는 여정"
7. **④ 아웃트로 배경** 위에 `amber-logo-light.png` + `GSITM 로고.svg` 를 '×'로 락업 + 자막 "Digital Renewal Proposal"

- 씬 사이 이음새엔 **② 트랜지션 스팅** 삽입.
- **자막은 전부 오버레이 텍스트 레이어**로 (`script.md` 문구 그대로). Veo로 글자 생성 금지.
- Flow 자막에서 한글이 어색하면, 마무리로 CapCut / Google Vids에서 자막 패스만 얹기.

## 컷 리듬 & 트랜지션
- 전체 40초, 컷 7개. S4(가치제안)에 9초로 가장 길게 — 광고의 핵심 훅.
- 트랜지션: 크로스디졸브 + 위치 이동(슬라이드) 중심. 하드컷은 S6 모바일 몽타주에서만 스냅하게.
- 카메라: 실제 화면은 정적 캡쳐/녹화 → 영상 AI가 줌/팬/틸트로 무빙 부여.

## 음악 싱크 포인트
- 0:00 인트로 스팅 → 0:03 비트 인 → S4(0:14) 빌드업 → **0:20 '338,000원' 확정에 임팩트 히트** → 0:35 아웃트로 리졸브.

## 자막 스타일 가이드
- 폰트: 국문 Pretendard SemiBold, 강조 숫자는 Playfair Display 또는 굵은 산세리프.
- 컬러: 크림(#f7f4ee) 텍스트 / 강조는 골드(#b7935b). 딥다크 배경 위 대비 확보.
- 등장: 마스크 슬라이드업 + 살짝 트래킹(자간) 축소. 화면당 1줄, 최대 12자 내외 유지.

## 비고 (녹화 시 반영할 앱 동작 특성)
- 인메모리 세션이라 **풀 리로드 시 로그아웃** → 녹화는 인앱 링크/버튼으로만 이동.
- 로그인은 아무 값이나 입력해도 홍길동/GOLD 세션 생성. 혜택 단계 진입 시 자동 복귀.
- 결제 버튼 클릭 후 **약 1초 로딩** 뒤 완료 화면으로 이동(로딩 상태도 녹화 소스로 유용).

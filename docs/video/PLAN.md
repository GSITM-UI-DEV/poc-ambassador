# 제안 시연 영상 제작 플랜

**THE AMBASSADOR SEOUL × GSITM — Digital Renewal Proposal Film**

프로토타입(React 앱) 화면을 기반으로 한 **"시연 + 광고"** 톤의 40초 영상 제작 플랜.
실제 영상 편집/렌더링은 **영상 전문 AI**에 맡기고, 이 문서는 그 AI에게 넘길 **소재·설계·대본**을 정리한다.

---

## 0. 컨셉 & 레퍼런스

### 레퍼런스 (사용자 지정)
전부 **Google 제품 런칭 필름** — 깔끔한 UI 쇼케이스 톤.

| # | 영상 | 차용할 요소 |
|---|---|---|
| 1 | Create interactive visuals with Gemini 3 in Google Search | 플로팅 UI 패널, 부드러운 줌/팬, 인터랙션 하이라이트 |
| 2 | New in Search \| Personal Intelligence in AI Mode | 키네틱 타이포 자막, 개인화/관계 서사 |
| 3 | Gemini 3 Flash: Frontier intelligence, built for speed | 스냅한 컷 전환, 스피드감, 미니멀 |

### 우리 버전의 방향
> **"Google 필름의 깔끔한 모션·구성 + 앰배서더의 럭셔리 골드 팔레트"**

- Google 레퍼런스는 밝고 컬러풀 → 우리는 **골드/크림/딥다크의 절제된 럭셔리 톤**으로 치환
- 화면(UI)이 주인공. 사람/호텔 실사 대신 **프로토타입 화면 자체를 스타일리시하게 연출**
- 자막(키네틱 타이포)로 핵심 메시지 전달, 배경음악은 앰비언트+업비트

### 핵심 메시지 (제안서 3축)
1. **가치제안** — 정가/회원가/할인율/절감액 동시 노출, GOLD 15% 배지, 공식채널 전용가
2. **고객경험** — 목적 기반 탐색(기념일·가족·미식), 객실+다이닝 원스톱 예약, 단계 최소화
3. **고객관계** — 멤버십 대시보드(포인트·등급 게이지·소멸 예정), 생애주기 추천

## 3. 시연 시나리오 (소스 화면)

앱은 로컬 `npm run dev` (http://localhost:5173) 기준. 라우팅으로 PC/모바일 분리.

**PC 여정** `/`
```
메인 → 검색결과(/search) → 호텔상세(/hotels/:id)
     → 예약 3-step(옵션→혜택→결제, /booking) → 완료(/booking/complete) → 마이페이지(/mypage)
```
**모바일 여정** `/mobile`
```
홈 → 검색 → 상세 → 옵션추가 → 회원혜택 적용 → 결제 → 예약완료
```

**가격 검증치(결제 화면):** 정가 345,000 + 케이크 35,000 − 회원 32,000 − 쿠폰 10,000 = **338,000원**, 적립 **3,200P**
→ 이 숫자 애니메이션이 §5 S4(가치제안)의 핵심 비주얼.

**로그인:** 아무 값 입력 → 홍길동 / GOLD 세션 생성 (혜택 적용 단계는 로그인 필요).

## 4. 워크플로우 (단계별)

### Phase 1 — 화면 캡쳐 (스틸)
- `npm run dev` 실행 후 Playwright로 각 라우트 방문 → 풀페이지/뷰포트 PNG 캡쳐
- PC 1920×1080 (필요시 풀페이지 스크롤샷), 모바일 390px 뷰포트 → 1080폭 리타이밍
- 로그인·혜택적용 등 상태별 캡쳐 포함 (before/after 가격 블록 등)

### Phase 2 — 시연 영상 녹화 (raw 클립)
- Playwright 영상 녹화로 **실제 조작 흐름**을 클립화 (hover, 스크롤, 단계 이동, 가격 변동)
- 브라우저 크롬(주소창 등) 없는 깨끗한 뷰포트로 녹화 → 영상 AI가 프레임/모션 얹기 좋게
- 씬 단위로 짧게 끊어 저장 (편집 자유도↑)

### Phase 3 — 스토리보드 + 스크립트
- §5 / §6 확정 → `storyboard.md`, `script.md`로 분리 저장

### Phase 4 — 영상 AI 전달 패키지
- §7 명세대로 stills/clips/브랜드에셋/대본/음악레퍼런스 묶어 전달

## 7. 영상 AI 전달 패키지 (§Phase 4 상세)

영상 AI에게 아래를 **한 폴더로** 넘긴다.

1. **화면 스틸** (`stills/`) — 씬별 고해상도 PNG, 파일명에 씬번호 매핑
2. **시연 클립** (`clips/`) — 브라우저 크롬 없는 조작 녹화 MP4
3. **스토리보드** (`storyboard.md`) — 타이밍·자막·모션 지시
4. **스크립트** (`script.md`) — 자막 텍스트 + (선택)VO 원고
5. **브랜드 에셋**
   - 로고 원본 (⚠️ 너가 제공 — 실제 앰배서더 로고 파일)
   - 컬러 토큰: 골드 `#b7935b` / 골드라이트 `#d4bc8b` / 딥다크 `#1c1b19` / 크림 `#f7f4ee` / 골드세리프 `#a8834e`
   - 폰트: 본문 Pretendard, 디스플레이 Playfair Display
6. **음악·무드 레퍼런스** — 위 Google 영상 3개 URL + "앰비언트 + 절제된 업비트, 럭셔리·모던" 무드 노트
7. **스펙 시트** — 화면비/해상도/길이/자막스타일 (§8 확정값)

## 8. 결정 필요 사항 (너가 정해줘야 최종 확정 가능)

| # | 항목 | 옵션 / 기본값 제안 |
|---|---|---|
| A | **화면비** | 16:9 (제안 PT 상영·유튜브) **[기본]** / 세로 9:16 추가 여부 |
| B | **내레이션** | 자막+음악만 / 한국어 성우 VO 추가 (Google 레퍼는 대체로 VO 有) |
| C | **언어** | 한국어 단독 / 한·영 병기 |
| D | **로고 원본** | 실제 앰배서더 로고 파일 있는지 (저작권 사용 가능 여부) |
| E | **음악** | 레퍼런스 무드로 AI 생성 / 특정 라이선스 곡 지정 |
| F | **용도·플랫폼** | 제안 PT 상영 / 유튜브 업로드 / 둘 다 → 해상도·길이 미세조정 |
| G | **사용할 영상 AI** | 어떤 툴인지 (Veo / Runway / Sora 등) → 입력 포맷·클립 길이 제약 확인 |

> **바로 진행 가능한 기본값:** A=16:9, B=자막+음악(VO는 나중에 얹기 가능), C=한국어, E=레퍼 무드.
> D(로고)·G(영상 AI 종류)는 확정될수록 결과물 품질이 올라감.
